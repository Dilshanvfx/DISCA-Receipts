# DISCA Receipt Management Engine

A professional receipt generation and management system for DIS Cricket Academy.

## Project Structure

- `/frontend`: React + Vite + Tailwind CSS application (PWA enabled).
- `/backend`: Node.js + Express server for handling data (optional).

## Getting Started

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev` (for development)
4. `npm run build` (for production/PWA)

### Backend
1. `cd backend`
2. `npm install`
3. `node server.js`
